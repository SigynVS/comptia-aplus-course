Software Troubleshooting

Diagnosing Windows OS problems, application issues, malware symptoms, and mobile OS/app/security trouble — and the right fix for each.

3.1 Windows OS problems

SymptomCommon fix
Blue Screen (BSOD) / stop errorNote the code; check recent drivers/hardware/RAM; boot Safe Mode; roll back.
Boot problems / no OS foundCheck boot order, run Startup Repair, bootrec, verify drive.
Frequent shutdowns / lockupsOverheating, drivers, failing hardware, malware.
Slow performance / high resource usageTask Manager → end runaway process; disable startup items (msconfig); check disk/malware.
Slow bootTrim startup apps, check disk health, malware scan.
Services not startingservices.msc; check dependencies & Event Viewer.
Time driftSync time; check CMOS battery.
Profile/USB issuesRebuild profile; reinstall driver.

Common tools & repair steps

  • Safe Mode, Last Known Good, System Restore, Startup Repair, sfc /scannow, chkdsk, DISM, reimage/reinstall, update drivers, roll back updates.
  • Boot order: least invasive first (reboot → Safe Mode → restore point → repair → reinstall).
On the exam: Choose the least invasive effective fix. Try a restore point or driver rollback before a full reinstall.

3.2 Application problems

  • App crashes / won't open — update or repair the app, run as admin, check compatibility mode, reinstall.
  • Compatibility — verify OS/architecture (32 vs 64-bit) and minimum requirements.
  • Slow / unresponsive — insufficient RAM/CPU; close background apps; check for updates.
  • Permission / "access denied" — UAC, file/folder permissions, run as administrator.

3.3 Malware symptoms on a workstation

  • Pop-ups, browser redirection, changed home page, rogue antivirus, new toolbars.
  • Slow system, high network/CPU, disabled security tools, missing files.
  • OS update failures, certificate warnings, altered hosts file, unexpected DNS.
  • Fix with the 7-step malware removal process (Security domain): identify → quarantine → disable System Restore → remediate → schedule scans/updates → re-enable System Restore → educate user.
Tip: Browser redirects + altered hosts file = classic malware. Reset the browser and clean the hosts file as part of remediation.

3.4 Mobile OS & app issues

SymptomFix
App won't launch / crashesForce-stop, clear cache/data, update, reinstall.
Slow performance / freezingClose apps, free storage, reboot, check updates.
Battery drains fast / overheatsFind rogue app, lower brightness, disable background refresh, replace battery.
No connectivity / dropped signalToggle airplane mode, re-pair Bluetooth, reset network settings.
Can't update / installFree storage, verify OS version, check account/store.

3.5 Mobile security symptoms

  • High resource use, unexpected data usage, limited connectivity, fake/unauthorized apps.
  • Leaked personal files/data, account access issues, signal interference.
  • Responses: uninstall suspicious apps, run mobile anti-malware, update OS, reset to factory, review app permissions, enable screen lock & remote wipe.
On the exam: Unexpected high data usage + a recently installed app often means leaked data / malware — uninstall and scan.
