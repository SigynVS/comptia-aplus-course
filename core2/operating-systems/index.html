Operating Systems

Windows editions and tools, the command line, system configuration utilities, OS installation, and a tour of macOS, Linux, and mobile OSes.

1.1 Windows editions & features

EditionNotes
HomeConsumer; no domain join, no BitLocker, no Group Policy (gpedit).
ProAdds domain join, BitLocker, Group Policy, RDP host.
Pro for Workstations / EnterpriseMore cores/RAM, advanced management features.
  • 32-bit (x86) caps RAM at ~4 GB; 64-bit (x64) supports far more.
  • Domain vs Workgroup: domain = central authentication (Active Directory); workgroup = peer-to-peer.

1.2 Command-line tools

CommandUse
ipconfigView/release/renew IP config (/all, /release, /renew, /flushdns).
pingTest reachability.
tracertTrace the route to a host.
nslookupQuery DNS.
netstatActive connections/ports.
chkdskCheck/repair disk (/f fix, /r recover).
sfc /scannowSystem File Checker — repair OS files.
diskpartPartition management.
gpupdate / gpresultApply / view Group Policy.
format / copy / robocopy / xcopyDisk & file operations.
shutdownShut down/restart from CLI.
On the exam: sfc /scannow repairs corrupted system files; chkdsk /f fixes file-system errors; ipconfig /flushdns clears the DNS cache.

1.3 Windows tools & the Control Panel / Settings

  • Task Manager — processes, performance, startup, services, users.
  • MMC snap-ins: devmgmt.msc (Device Manager), diskmgmt.msc (Disk Management), services.msc, compmgmt.msc, eventvwr.msc (Event Viewer), perfmon, taskschd.msc, certmgr.msc, lusrmgr.msc.
  • System Configuration msconfig, Registry Editor regedit, Resource Monitor resmon, dxdiag.
  • Control Panel applets: Internet Options, Devices & Printers, Programs & Features, Network & Sharing, System, Windows Defender Firewall, Power Options, User Accounts, Indexing, File Explorer Options, Admin Tools.

1.4 Disk & OS installation

  • Partition styles: MBR (≤2 TB, 4 primary partitions) vs GPT (huge disks, 128 partitions, needs UEFI).
  • File systems: NTFS (Windows; permissions, encryption, journaling), FAT32 (4 GB file limit, broad compatibility), exFAT (large files on removable media), ext4 (Linux), APFS (macOS).
  • Install types: clean, in-place upgrade, image deployment, repair, recovery, unattended, network/PXE.
  • Boot methods: USB, optical, PXE, internet-based.

1.5 Networking config in Windows

  • Configure IP (static vs DHCP), DNS, gateway, proxy, VPN, alternate config.
  • Workgroup vs domain join; network locations (Public/Private) change firewall behavior.
  • Establish network shares, mapped drives, and printer connections.

1.6 Other operating systems

  • macOS: Finder, Mission Control, Spotlight, Keychain, Time Machine (backup), Disk Utility, FileVault (encryption), Gatekeeper.
  • Linux: shell commands — ls, pwd, cd, cp, mv, rm, chmod, chown, sudo, apt/yum, grep, ps, df.
  • Mobile OSes: iOS (closed), Android (open), updates, app stores.
