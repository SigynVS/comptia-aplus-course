Network Implementation

Routing and switching technologies, VLANs, wireless standards, and the physical build.

2.1 Routing technologies

  • Static routes (manual) vs dynamic routing protocols.
  • Protocols: OSPF (link-state, IGP), EIGRP (Cisco hybrid), BGP (path-vector, the internet's routing protocol), RIP (legacy distance-vector).
  • Default route (0.0.0.0/0), administrative distance, metrics, NAT/PAT, and FHRP (e.g., VRRP) for gateway redundancy.

2.2 Switching technologies

  • VLANs segment a LAN logically; 802.1Q tags trunk traffic between switches.
  • STP (Spanning Tree) prevents switching loops.
  • Port security, MAC tables, port aggregation (LACP), PoE, and jumbo frames.
  • Trunk port (carries many VLANs) vs access port (one VLAN).
On the exam: A trunk (802.1Q) carries multiple VLANs between switches; an access port belongs to a single VLAN for an end device.

2.3 Wireless standards

802.11nWi-Fi 42.4 & 5 GHz

  • Concepts: SSID, channels (2.4 GHz: 1/6/11), channel width, MIMO/MU-MIMO, band steering.
  • Antenna types: omnidirectional vs directional; site survey & heat map for placement.

2.4 Physical installations

  • Cabling: Cat 5e/6/6a/7/8 twisted pair, single-mode vs multimode fiber, coax.
  • Connectors: RJ45, LC/SC/ST (fiber), F-type.
  • T568A/T568B wiring; straight-through vs crossover; patch panels, 110/66 blocks.
  • Hardware: switches, routers, APs, firewalls, NIC; environment: power (UPS/PDU), cooling, rack units.
Tip: Use single-mode fiber for long distance, multimode for shorter high-bandwidth runs within a building.
