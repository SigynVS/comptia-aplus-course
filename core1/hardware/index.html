Core 1 · 220-1101 · Domain 3.0 (25%)

Hardware

The biggest Core 1 domain: cables and connectors, memory, storage, motherboards, CPUs, power supplies, and peripherals.

3.1 Cables & connectors

CableNotes
Cat 5e / Cat 6 / Cat 6aTwisted-pair Ethernet. 5e=1 Gbps; 6=1–10 Gbps (short); 6a=10 Gbps. RJ45 connector.
Coaxial (RG-6)Cable internet/TV. F-type connector.
Fiber (SMF/MMF)Light over glass; immune to EMI, long distance.
USB 2.0 / 3.x2.0=480 Mbps; 3.2 Gen 1=5 Gbps; Gen 2=10 Gbps. USB-C is reversible.
SATAInternal drives — 7-pin data + 15-pin power.
Lightning / Molex / DB / PunchdownApple legacy / old peripheral power / serial / 110 block.
  • Video: HDMI, DisplayPort, DVI, VGA (analog, legacy).
  • RJ45 = Ethernet (8 wires); RJ11 = phone/DSL (smaller).

3.2 RAM

TypeNotes
DDR3 / DDR4 / DDR5Not interchangeable — different notch keys & voltages. DDR5 is current.
DIMM vs SODIMMDesktop vs laptop form factor.
ECCError-Correcting Code — detects/corrects errors; used in servers.
ChannelsSingle/dual/triple/quad — install in matched pairs for dual-channel bandwidth.

3.3 Storage devices

  • HDD — spinning platters; speeds 5400 / 7200 / 10000+ RPM.
  • SSD — flash; forms: 2.5" SATA, M.2 (SATA or NVMe), mSATA.
  • NVMe over PCIe is the fastest mainstream storage.
  • Optical: CD/DVD/Blu-ray (legacy).
  • RAID: 0 (stripe, speed, no redundancy), 1 (mirror, redundancy), 5 (stripe + parity, needs 3+ disks), 10 (mirror + stripe).
On the exam: RAID 0 has no fault tolerance. RAID 1 mirrors. RAID 5 survives one disk failure. RAID 10 needs four disks.

3.4 Motherboards, CPUs & add-on cards

  • Form factors: ATX > microATX > Mini-ITX.
  • Expansion: PCIe (x1, x4, x8, x16). GPUs use x16.
  • CPU sockets: Intel LGA (pins on socket), AMD historically PGA (pins on chip; AM5 is LGA).
  • Cooling: heat sink + fan, thermal paste, liquid cooling.
  • BIOS/UEFI: boot order, Secure Boot, TPM, passwords. CMOS battery (CR2032) keeps settings/clock — a dead one resets the date/time.

3.5 Power

  • PSU wattage must exceed total load; rails: +3.3V, +5V, +12V.
  • 24-pin main + 4/8-pin CPU (EPS) + PCIe power for GPUs.
  • Know input voltage (115V vs 230V) and dual-rail concerns.

3.6 Peripherals & printers (overview)

  • Multifunction devices, ADF/flatbed scanners, monitors, KVM.
  • Printer types: laser (7-step imaging: processing, charging, exposing, developing, transferring, fusing, cleaning), inkjet, thermal, impact (multipart forms), 3D.
On the exam: Memorize the laser imaging process order and that impact printers are used for carbon-copy/multipart forms.
